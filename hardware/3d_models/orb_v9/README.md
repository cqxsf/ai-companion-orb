# Orb V9 — 工业设计级 3D 模型

> 从"概念球体"到"可量产桌面情绪物件"的跨越。

---

## 核心设计参数

| 参数 | V8 | **V9** | 改进 |
|------|----|--------|------|
| 直径 | Ø98mm | **Ø112mm** | 更有桌面存在感 |
| 高度 | 68mm | **86mm** | 内部空间更充裕 |
| 比例 H/W | 0.69 | **0.77** | 蛋形，不像音箱 |
| 曲面 | 三段拼接 | **连续椭球** | 完全无拼接感 |
| 顶盖 | 光扩散罩 | **浮岛结构** | 0.6mm 缝隙=呼吸口 |
| 灯环 | 可见灯带 | **隐藏扩散罩** | 光从内部自然透出 |
| 零件数 | 7+ | **5** | 装配更简单 |

---

## 椭球方程

$$\left(\frac{x}{56}\right)^2 + \left(\frac{z}{43}\right)^2 = 1$$

绕 Z 轴旋转生成完整椭球体，曲率从顶部到底部连续过渡，无断层。

## 文件结构

### FreeCAD 版本（推荐，参数化可修改）
| 文件 | 说明 |
|------|------|
| `orb_v9_config.py` | 全局参数定义（Python） |
| `orb_v9_freecad.py` | FreeCAD 建模主脚本（一键生成 5 零件） |

### OpenSCAD 版本（与 V4-V8 工具链兼容）
| 文件 | 说明 |
|------|------|
| `orb_config.scad` | 全局参数 + `v9_r(z)` 椭球函数 |
| `orb_shell.scad` | 椭球壳体模块 |
| `orb_shell_top.scad` | 上壳 |
| `orb_shell_bottom.scad` | 下壳（带螺丝柱） |
| `orb_led_ring.scad` | LED 灯环扩散罩 |
| `orb_top_cap.scad` | 浮岛顶盖（带麦克风孔） |
| `orb_base.scad` | 底盖（带防滑垫定位槽） |
| `orb_assembly.scad` | 装配预览（支持爆炸视图） |
| `export_stl.sh` | 一键 STL 导出脚本 |

## 快速开始

### FreeCAD 建模
```bash
# 1. 安装 FreeCAD
brew install --cask freecad

# 2. 在 FreeCAD 中执行
# Macro → Execute Macro → orb_v9_freecad.py
# 自动生成 5 个零件 + 导出 STL
```

### OpenSCAD 导出
```bash
cd hardware/3d_models/orb_v9
bash export_stl.sh
```

## 装配说明

1. **下壳 + 底盖** → M2×6 螺丝 ×4 从底部固定
2. **LED 灯环** → 卡入下壳上沿环形槽
3. **上壳** → 扣合到灯环上沿（环形卡扣）
4. **顶盖** → 浮放于上壳顶部（0.6mm 自然缝隙）
5. 测试：灯环亮起后，光线应从缝隙和灯环均匀透出
